# WeGoTickets

Just a simple scraper. :D

## Dependencies 

 * glide package manager
 * go compiler

## Run

Clone the project.

```
$ git clone https://github.com/patito/wegotickets.git
```

Building the project.

```
$ make
```

Running...

```
$ make run
```
